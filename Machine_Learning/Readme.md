# Machine Learning

This folder contains various implementations of machine learning algorithms, divided into Supervised and Unsupervised learning techniques.

## 📂 Folders

### 🔵 [Supervised Learning]
Models trained on labeled data to predict specific outputs.
*   **Simple Linear Regression**: Modeling the relationship between two continuous variables.
*   **Multiple Linear Regression**: Predicting an outcome based on multiple predictor variables.
*   **Multiple Linear Regression using RFE**: Using Recursive Feature Elimination to select the most significant features for the regression model.
*   **Decision Tree**: A tree-structured model used for making predictions through logic-based splits.
*   **Ensembles & Random Forests**: Advanced models that combine multiple learners (like trees) to improve overall predictive performance and stability.

### 🟢 [Unsupervised Learning]
Models that find hidden patterns or groupings in data without pre-defined labels.
*   **Clustering**: Grouping data points together based on similarities (e.g., K-Means, Hierarchical Clustering).
